PulseCare 2026: Hospital Management System

PulseCare 2026 is a modern, fully functional desktop application designed to streamline hospital administrative operations. Developed as a final project for the Java Object-Oriented Designing and Programming course at Alliance University, it demonstrates the integration of advanced Swing UI techniques with robust OOP architecture.

# key features
Modern Dashboard: Includes a purple-to-pink gradient hero banner, real-time metric cards, and a department highlights section.Comprehensive Management: Dedicated modules for Patient Registration, Doctor Directory, and Appointment Scheduling.Live Analytics: A "Shift Snapshot" sidebar provides real-time updates on total patients, doctors, appointments, and occupancy rates.Advanced Filtering: Support for regex-based live searching and categorical filtering (e.g., by gender or department) across all data tables.Visual Indicators: Color-coded status badges for appointments (Scheduled, Pending, Completed) and striped table rows for better readability.Data Validation: Built-in form validation to ensure data integrity and prevent empty fields or invalid numeric inputs


### Technologies Used
The project strictly adheres to Object-Oriented Programming (OOP) principles, including encapsulation, inheritance, polymorphism, and abstraction
